## Быстрый старт

### Настройка окружения

1. **Создайте виртуальное окружение Python:**
   ```batch
   # Windows
   setup_venv.bat
   
   # Linux/macOS
   chmod +x setup_venv.sh
   ./setup_venv.sh
   ```

2. **Используйте обертки для pip и python:**
   ```batch
   # Windows
   pip.bat install package_name
   python.bat script.py
   
   # Linux/macOS
   ./pip.sh install package_name
   ./python.sh script.py
   ```

Подробнее см. [README_VENV.md](README_VENV.md) и [CLION_SETUP.md](CLION_SETUP.md).
